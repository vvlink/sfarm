# 使用说明

当有嘉宾来参观智慧农场系统的，如何快速展示这一系统物联功能？

在服务器上运行该程序，然后将地址做成二维码。用手机扫码（手机要能够访问服务器）的方式，打开一个网页，就能看到数据，并且控制浇水。

## 版本

1.1版，实现对多个设备的数据监视和浇水控制，使用下拉方式切换；

1.0版，实现对一个设备的数据监视和浇水控制；

## 搭建说明

1.安装支持库（remi、siot）；

pip install remi

pip install siot

2.手机要能够访问siot服务器。

3.本程序采用remi库编写。

github：https://github.com/dddomodossola/remi
